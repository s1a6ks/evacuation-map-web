# 🏢 Evacuation Map Web System

**Інтелектуальна система проектування та аналізу планів евакуації на основі графових моделей.**

Проект дозволяє не просто малювати плани приміщень, а створювати цифрову модель будівлі, яка автоматично аналізує безпеку та прораховує найгірші сценарії евакуації згідно з державними будівельними нормами.

---

## 🔥 Ключовий функціонал

### 1. 🏗️ Інтерактивний редактор планів
*   **Vector Drawing**: Малювання стін, дверей та вікон за допомогою 2D Canvas.
*   **Multi-floor Support**: Підтримка декількох поверхів зі зв'язками через сходи.
*   **Grid Snapping**: Точне позиціонування елементів за сіткою.
*   **Room Detection**: Автоматичне розпізнавання замкнених просторів (кімнат) та обчислення їх площі.

### 2. 🧠 Графовий двигун аналізу
*   **Auto-Graph Generation**: Система перетворює малюнок у математичний граф прохідності.
*   **Pathfinding Optimization**: Використання алгоритму **Dijkstra** для миттєвого пошуку найкоротшого шляху до виходу з будь-якої точки.
*   **Real-time Metrics**: Розрахунок відстані в метрах та прогнозованого часу евакуації (сек/хв) з урахуванням швидкості руху людей.

### 3. 🛡️ Перевірка за ДБН В.1.1-7
Система автоматично підсвічує порушення пожежної безпеки:
*   🔴 **Тупикові зони**: Виявлення кімнат без альтернативних шляхів виходу.
*   🔴 **Недостатня кількість виходів**: Обчислення необхідної кількості евакуаційних виходів залежно від площі.
*   ⚠️ **Перевищення довжини шляху**: Контроль ліміту в 25 метрів (або іншого згідно з нормами).
*   🟠 **Bottlenecks**: Пошук "вузьких місць" у коридорах, де можливе утворення заторів.

### 4. 🎨 Сучасна візуалізація
*   **Dynamic UI**: Використання **GSAP** для плавних анімацій панелей та переходів.
*   **Interactive Analysis Panel**: Детальна таблиця з рейтингом безпеки кожної кімнати.
*   **Visual Indicators**: Кольорове маркування шляхів, виходів та небезпечних зон безпосередньо на плані.

| Складник | Технологія |
| :--- | :--- |

## ⚙️ Швидкий запуск

1.  **Клонуйте репозиторій**: `git clone ...`
2.  **Запустіть Frontend**:
    ```bash
    cd evacuation-frontend
    npm install
    npm run dev
    ```
3.  **Запустіть Backend**:
    *   Відкрийте `EvacuationSystem.sln` у Visual Studio.
    *   Натисніть `F5` або `dotnet run --project EvacuationSystem.Api`.
